An interactive, high performance web application designed to explore the Pokémon universe using real time data from the [PokeAPI](https://pokeapi.co/).

---

## ✨ Features

* **Live Search:** Find Pokémon instantly by name or index number.
* **Detailed Pokédex:** View comprehensive stats, abilities, and type advantages.
* **Responsive UI:** Optimized for everything from mobile screens to ultra-wide monitors.
* **High-Res Sprites:** Fetches official artwork and high-quality sprites.

## 🛠️ Tech Stack

| Technology | Purpose |
| :--- | :--- |
| **HTML5 / CSS3** | Structure and layout |
| **JavaScript (ES6+)** | Dynamic content and API fetching |
| **PokeAPI** | Source of all Pokémon data |

You can view this site at: https://hamzacyberpatcher.github.io/pokemon-web/
